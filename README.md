## 📋 Project Overview

The Book Haven is a full-stack web application that allows users to discover, share, and discuss books. Users can browse through a collection of books, add their own favorites, leave comments, and explore books. The application features a clean, modern interface with dark/light theme support.


## ✨ Key Features

### Core Features
- 🔐 **User Authentication**
  - Email/Password authentication
  - Google Sign-In integration
  - Private routes for authenticated users
  
- 📖 **Book Management**
  - Browse all books in an organized table view
  - Add new books with detailed information
  - Update your own books
  - Delete books you've added
  - View detailed book information
  
- ⭐ **Book Display**
  - Latest 6 books showcased on homepage
  - Sortable book table by rating (ascending/descending)
  - Star rating visualization
  - Genre categorization
  
- 💬 **Comment System**
  - Leave comments on any book
  - Delete your own comments
  - View all user comments with timestamps
  - Real-time comment updates

- 🎨 **User Interface**
  - Responsive design (mobile, tablet, desktop)
  - Dark/Light theme toggle
  - Smooth animations with Framer Motion
  - Professional shadow effects

### Additional Features
- 🖼️ **Image Upload** - Upload book covers via ImgBB
- 🔍 **My Books** - Personal collection management
- 📊 **Table View** - All books and user books displayed in tables
- 🎯 **Form Validation** - React Hook Form with comprehensive validation
- 🍞 **Toast Notifications** - SweetAlert2 for user feedback

## 🛠️ Tech Stack

### Frontend
- **Framework:** React with Vite
- **Routing:** React Router
- **Styling:** Tailwind CSS + DaisyUI
- **State Management:** TanStack Query
- **Form Handling:** React Hook Form
- **HTTP Client:** Axios
- **Animations:** Framer Motion
- **Notifications:** SweetAlert2
- **Icons:** React Icons

### Backend
- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB Atlas
- **Authentication:** Firebase
- **Middleware:** CORS

### Additional Services
- **Image Hosting:** ImgBB API
- **Authentication:** Firebase Authentication
- **Deployment:** 
  - Frontend: Firebase Hosting
  - Backend: Vercel
